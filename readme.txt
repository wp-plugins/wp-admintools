== Description ==

WP AdminTools allows comprehensive SEO optimization, cleaning up the database and many useful features that extend your WordPress Blog. Get all SEO features you need with this plugin. 

Some features in WP AdminTools include:

* Disable Revisions or Trash Feature on your Blog to optimize your Database
* You can manually set the Auto Save Interval
* Restrict your Blogs search results to specific posts or pages
* Let the PlugIn automatically generate optimized SEO Titles
* Set customized meta information like description, keywords or robots information (index,noindex)
* Automaticcaly exlude pages from being indexed by search engines to avoid duplicate content
* "Clean-Up" your <head> information
* Trash all open comments or spam comments with a single click
* and much more ...

== Installation ==

Upload the plugin to your blog, Activate it and configure to your needs.

You're done!
